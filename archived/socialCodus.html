About socialCodus

socialCodus is a social media network for Epicodus students with full CRUD (Create, Read, Update, Delete) capabilities. The web app was built with C# and SQL on the back end and CSS/Sass and Bourbon & Neat on the front end. socialCodus members are able to post status updates, friend/unfriend each other, leave comments on each others' profiles, like/dislike posts and comments, and see a feed of their friends' most recent posts on the homepage. socialCodus was built within 3 1/2 days.

Process

Coding can be stressful, so having a welcoming support network can be helpful. We wanted users to feel calm and relaxed when accessing socialCodus, so we gave the app's branding shades of green and blue, implemented smooth CSS transitions throughout the user interace, and added a calming and zen-like animation of CSS triangles that cascaded down the screen.

socialCodus' branding:

I worked on all the front end aspets of the application with Arianna Chau. One of the challenges that we faced when working on socialCodus was how to structure and set up our files so they would be compatible with our back end team (this meant lots of comments in our code and frequent communication with them). We checked in with our back end team first thing every morning and every couple hours. This made certain that we were on track to hit our MVP, that the files we were handing over contained exactly what they needed, and we gave them a heads up before we merged our feature branches so we could work out any potential conflicts together.
