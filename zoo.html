View Code

About Rose City Zoo

Rose City Zoo is a web application built with AngularJS and TypeScript with full CRUD (Create, Read, Update, Delete) capabilities. The site has two purposes: the first is to display the zoo's animal information in a child-friendly way, with bright colors, playful animations, and simple language. The second is for zoo staff to be able to manage the site's content, which includes editing animal data (name, age, likes, dislikes, etc.), as well as adding new and deleting animals from the application.

Process

Rose City Zoo was built within 8 hours after I had covered Angular concepts for 4 days. The app was developed before I knew that the best practice for structuring an Angular app was to give each component its own HTML, TypeScript, and CSS files, so all the HTML is written directly into the TypeScript files. The Angular structure of Rose City Zoo is as follows:

  • Model:
  • Animal (for animal information: species, name, age, diet, location, caretakers, sex, likes, dislikes, fun fact, and popularity)
  • Entry point & Root module:
  • main.ts
  • app.module.ts
  • Components:
  • app.component.ts
  • animal-inventory.component.ts
  • edit-animal.component.ts
  • footer-layout.component.ts
  • nav-layout.component.ts
  • Pipe:
  • sorter.pipe.ts (for sorting each animal by its popularity. Sorry Arctic Fox, you didn't make the cut!)
